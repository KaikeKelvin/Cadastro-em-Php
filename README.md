# Cadastro-em-Php
Sistema de Cadastro Full-Stack (PHP &amp; Supabase) CRUD completo desenvolvido com PHP 8 e PostgreSQL. Integra interface responsiva em Bootstrap 5 com persistência segura de dados na nuvem via Supabase. Foco em segurança: utiliza PDO contra SQL Injection e password_hash para proteção de senhas. Ideal para portfólio.

 📝 Sistema de Cadastro PHP + Supabase
Um sistema completo de gerenciamento de cadastros (CRUD) desenvolvido com PHP 8 e PostgreSQL (hospedado no Supabase). O projeto utiliza Bootstrap 5 para uma interface responsiva e moderna.

🚀 Funcionalidades
Criação: Cadastro de usuários com validação de campos obrigatórios.

Segurança: Criptografia de senhas com password_hash.

Leitura: Listagem dinâmica dos registros ordenados por ID.

Atualização: Edição completa de informações existentes.

Exclusão: Remoção de registros do banco de dados via PDO.
