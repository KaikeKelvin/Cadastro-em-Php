# Cadastro-em-Php
Sistema de Cadastro Full-Stack (PHP &amp; Supabase) CRUD completo desenvolvido com PHP 8 e PostgreSQL. Integra interface responsiva em Bootstrap 5 com persistência segura de dados na nuvem via Supabase. Foco em segurança: utiliza PDO contra SQL Injection e password_hash para proteção de senhas. Ideal para portfólio.
